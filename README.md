# 🔍 NPS 인사이트 Q&A

자연어 질문으로 NPS 데이터를 탐색하는 분석 도구

## ✨ 주요 기능

- 💬 **자연어 질문**: "인천팀에서 시니어 비중 높은 T크루는?"
- 🎛️ **스마트 필터**: 질문에서 조건 자동 추출
- ⚡ **즉시 분석**: API 비용 없이 실시간 처리
- 📊 **다양한 분석**:
  - 시니어 GAP 분석
  - 기간별 NPS 비교
  - 매장/대리점 비교
  - 트렌드 분석

## 🎯 질문 예시

### 기본 분석
- "시니어 비중 높은데 NPS 낮은 T크루"
- "NPS 87% 미만 매장"

### 기간 비교
- "9~11월 대비 12월 NPS 상승한 T크루"
- "12월 25일 기준 대비 NPS 하락한 매장"

### 매장/지역 분석
- "인천팀 큰곰마을점 NPS 분석"
- "대리점 D13422 인접 매장 비교"

## 🚀 로컬 실행

```bash
streamlit run app.py
```
